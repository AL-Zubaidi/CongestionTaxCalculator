To be able to test the Api you need to use Postman

1- Use this url with a Get method
https://localhost:7142/api?vehicle=6&dates=2013-08-19T17:01:28&dates=2013-08-19T17:30:29&dates=2013-08-19T18:07:29

2- Select any number between 0 to 5 as a tax free vehicle.

	  Motorcycle,
        Tractor,
        Emergency,
        Diplomat,
        Foreign,
        Military

3- Select number 6 as a "Car" as a not tax free vehicle.

4- change the dates if you want to try any other dates.
